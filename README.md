# Flickr Photo Stream
An app displaying the latest photo uploads from Flickr's public feed.

## How to run:

 1. Clone this repo ```git clone https://github.com/patrivet/flickr-photo-stream.git```
 2. Install dependancies ```npm install```
 3. Run app ```npm start```
